# Webots Simulation

This folder contains a minimal Webots setup for validating the quadcopter controller.

- controllers/quadcopter_controller.py — controller implementation
- worlds/quadcopter.wbt — minimal world file (placeholder)

To run:
1. Open Webots
2. Open `simulations/webots/worlds/quadcopter.wbt`
3. Run the simulation and ensure the controller is selected
